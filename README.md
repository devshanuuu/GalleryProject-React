# 📸 React Image Gallery

A simple React app that fetches and displays images using an API with pagination.

## 🚀 Features

- Fetch images from Picsum API  
- Display images in card layout  
- Pagination (Next / Prev buttons)  
- Loading state while fetching data  
- Reusable Card component  

## 🛠️ Tech Used

- React (useState, useEffect)  
- Axios (API calls)  
- Tailwind CSS (styling)
